Introduction / Business Problem¶

  • This project is intended to use a geo-spatial analysis in the city of Chicago to cluster neighborhoods to make a decision on where to open a new coffee shop.
  • A customer asks for a suggested location(s) near movie theater(s) and it is still underserved in a coffee retail segment.
  • I will attempt to cluster Chicago neighborhoods for the proposed coffee shop locations with an added analysis of low crime rates, as the customer would like to stay away from the rough neighborhoods.

Data Sources¶

  • Foursquare API : https://developer.foursquare.com/ : find venues and locations
  • Chicago Neighborhoods : https://data.cityofchicago.org/Facilities-Geographic-Boundaries/Boundaries-Neighborhoods/bbvz-uum9 - GeoJSON data
  • Chicago Crime Data 2018 - https://data.cityofchicago.org/Public-Safety/Crimes-2018/3i3m-jwuy - exported into CSV
  • 2010 Census Data Summarized to Chicago Community Areas - https://datahub.cmap.illinois.gov/dataset/2010-census-data-summarized-to-chicago-community-areas

Methodology¶

  • The 1st step will be to focus on areas with low crime rate per capita
    • We will work with both 'Chicago Crime Data' and 'Census Data', merge the 2 data sets to figure out what the crime rate per capita is in the Chicago community areas
      • There are 77 community areas in Chicago
    • Data cleaning will involve removing incomplete records, 'Domestic' reports, etc.
  • The 2nd step will be focusing on 20 community areas with the least CrimeRate
    • We will use the Foursquare API to find Coffee Houses close to Movie Theaters within a walking distance
      • 1500 m sounds like a good 'walking distance' definition, we will use that.
  • The final step will be k-Means Clustering, where we will analyze a resulting dataframe with prospective/recommended areas to open a new Coffee Shop

# common functions:
def get_category_type(row):
    try:
        categories_list = row['categories']
    except:
        categories_list = row['venue.categories']
        
    if len(categories_list) == 0:
        return None
    else:
        return categories_list[0]['name']

def getNearbyVenues(categoryID, names, latitudes, longitudes, radius):
    venues_list = []
    for name, lat, lng in zip(names, latitudes, longitudes):
        # create the API request URL
        url = 'https://api.foursquare.com/v2/venues/explore?&client_id={}&client_secret={}&v={}&ll={},{}&radius={}&limit={}&categoryId={}'.format(
            CLIENT_ID, 
            CLIENT_SECRET, 
            VERSION, 
            lat, 
            lng, 
            radius, 
            LIMIT,
            categoryID)
        #print(requests.get(url).json())
        try:
            # make the GET request
            results = requests.get(url).json()["response"]['groups'][0]['items']
        except:
            print('Your quota may have been exceeded')
            return
        
        # return only relevant information for each nearby venue
        venues_list.append([(
            name, 
            lat, 
            lng, 
            v['venue']['name'], 
            v['venue']['location']['lat'], 
            v['venue']['location']['lng'],  
            v['venue']['categories'][0]['name']) for v in results])

    nearby_venues = pd.DataFrame([item for venue_list in venues_list for item in venue_list])
    nearby_venues.columns = ['CommunityName', 
                  'CommunityNameLatitude', 
                  'CommunityNameLongitude', 
                  'Venue', 
                  'VenueLatitude', 
                  'VenueLongitude', 
                  'VenueCategory']
    
    return(nearby_venues)
